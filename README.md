#IoT based Smart Parking System
#Implemented Attributes
• Arduino Uno
• Esp8266 Wifi Module
• Servo
• IR Sensor
• ThingSpeak Cloud to view the data.
• Custom built website to control the whole iot system.
• Using IFTTT for sending mail to the user.
